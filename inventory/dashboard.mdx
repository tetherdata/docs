# Inventory Dashboard

The Inventory Dashboard is your central hub for monitoring inventory health, identifying issues, and taking action to prevent stockouts and overstocks.

<Note>
  The Inventory Dashboard is available to Admin users only. Sales Rep users should use the Demand Forecast views.
</Note>

## Accessing the Dashboard

Navigate to **Inventory** in the sidebar. The dashboard opens to the Overview tab by default.

## Dashboard Tabs

The Inventory page has four main tabs:

| Tab | Purpose |
|-----|---------|
| **Overview** | Alerts and health status summary |
| **Supply Plan** | Scheduled orders and recommendations |
| **Forecast** | Inventory projection charts |
| **Flow** | Visual replenishment diagram |

## Inventory Health Status

Each SKU/warehouse combination has a health status:

| Status | Meaning | Typical Action |
|--------|---------|----------------|
| **Healthy** | Stock levels are good | No action needed |
| **Low Inventory** | Below target but above safety stock | Plan replenishment |
| **Safety Stock Breach** | Below safety stock threshold | Order urgently |
| **Out of Stock** | Zero sellable inventory | Emergency order |
| **Overstock** | Above maximum threshold | Reduce orders, promote sales |

## Coverage Days

Coverage days indicate how long current inventory will last based on consumption forecasts.

### How It's Calculated

```
Coverage Days = Current Sellable Inventory / Daily Consumption Rate
```

### Interpreting Coverage Days

| Coverage | Interpretation |
|----------|----------------|
| **< 7 days** | Critical - immediate action needed |
| **7-14 days** | Urgent - order now |
| **15-30 days** | Plan replenishment |
| **31-60 days** | Healthy range (varies by product) |
| **> 60 days** | Consider if overstock |

<Info>
  Ideal coverage days vary by product and lead time. A product with 60-day lead time needs more coverage than one with 7-day lead time.
</Info>

## Inventory States

Tether tracks inventory in different states:

| State | Description |
|-------|-------------|
| **Sellable** | Available for sale |
| **On Hand** | Physically in warehouse |
| **Committed** | Reserved for orders |
| **Incoming** | In transit or on order |
| **Damaged** | Not available for sale |

### Understanding the Relationship

```
On Hand = Sellable + Committed + Damaged
Available to Promise = Sellable + Incoming
```

## Filtering the Dashboard

### By Warehouse

Focus on specific warehouses:

1. Click the warehouse filter
2. Select one or more warehouses
3. All data updates to show only selected locations

### By Collection

Filter by product group:

1. Open the collection filter
2. Select collections to include
3. Alerts and metrics update accordingly

### By Time Horizon

Change the projection window:

| Option | Shows Alerts For |
|--------|------------------|
| **30 days** | Short-term issues |
| **60 days** | Medium-term planning |
| **90 days** | Standard planning horizon |
| **180 days** | Extended view |
| **360 days** | Annual planning |

<Tip>
  Start with 90 days for daily operations. Use longer horizons for strategic planning sessions.
</Tip>
