# Supply Plan

The Supply Plan is where you manage replenishment orders‚Äîboth system-recommended and user-created. It's your action center for keeping inventory at optimal levels.

## Accessing the Supply Plan

Navigate to **Inventory** ‚Üí **Supply Plan** tab.

## Understanding the Supply Plan

The Supply Plan shows all scheduled replenishment orders:

- **System Recommendations** - Orders Tether suggests based on forecasts
- **User-Created Orders** - Orders you've manually added
- **Draft Transactions** - Orders in preparation

## Viewing Scheduled Orders

The Supply Plan displays orders in a table format:

| Column | Description |
|--------|-------------|
| **Order Date** | When the order should be placed |
| **Type** | Purchase, Transfer, or Production |
| **Source** | Supplier or source warehouse |
| **Destination** | Destination warehouse |
| **Alert Level** | Urgency (Urgent, Important, Normal) |
| **Status** | Scheduled, Draft, In Progress |
| **Line Items** | Number of SKUs in the order |
| **Total** | Order value (if applicable) |

### Order Sources

Orders are marked by their creation source:

| Badge | Meaning |
|-------|---------|
| **System** | Generated by Tether's recommendation engine |
| **User** | Manually created by a user |

## System Recommendations

Tether continuously analyzes your inventory and demand to recommend optimal orders.

### How Recommendations Work

The recommendation engine considers:

1. **Current Inventory** - Stock levels across warehouses
2. **Demand Forecast** - Projected consumption
3. **Lead Times** - Supplier and transfer times
4. **Safety Stock** - Target buffer levels
5. **Order Constraints** - Minimum quantities, lot sizes

### Viewing Recommendation Details

Click on a recommended order to see:

- **SKU Details** - What to order and quantities
- **Timing Logic** - Why this date was chosen
- **Impact Analysis** - How this order affects inventory
- **Alternative Options** - Other suppliers or timing

## Working with Orders

### Adding Orders to Supply Plan

For system recommendations:

<Steps>
  <Step title="Review the Recommendation">
    Click on a recommended order to see details
  </Step>

  <Step title="Verify Details">
    Check quantities, timing, and supplier selection
  </Step>

  <Step title="Add to Plan">
    Click **Add to Supply Plan** to accept the recommendation
  </Step>

  <Step title="Order Added">
    The order moves to your active supply plan
  </Step>
</Steps>

### Creating Drafts from Recommendations

Convert a recommendation directly to a transaction:

<Steps>
  <Step title="Select the Order">
    Click on the scheduled order
  </Step>

  <Step title="Click Create Draft">
    Select **Create Draft Transaction**
  </Step>

  <Step title="Review Draft">
    The draft opens with pre-filled details
  </Step>

  <Step title="Submit When Ready">
    Submit the draft to begin processing
  </Step>
</Steps>

### Bulk Actions

Work with multiple orders at once:

<Steps>
  <Step title="Select Orders">
    Use checkboxes to select multiple orders
  </Step>

  <Step title="Choose Bulk Action">
    Options include:
    - **Add All to Plan** - Accept multiple recommendations
    - **Remove from Plan** - Remove selected orders
    - **Create Drafts** - Convert all to transaction drafts
    - **Delete** - Remove orders entirely
  </Step>

  <Step title="Confirm">
    Review and confirm the bulk action
  </Step>
</Steps>

### Modifying Orders

Before an order becomes a submitted transaction, you can modify it:

**Adjust Quantities:**
1. Open the order details
2. Edit line item quantities
3. Save changes

**Change Timing:**
1. Open the order
2. Modify the order date
3. Save

**Add/Remove SKUs:**
1. Open line items
2. Add new SKUs or remove existing ones
3. Save changes

### Removing Orders

To remove an order from your supply plan:

1. Select the order
2. Click **Remove from Plan** or **Delete**
3. Confirm the action

<Warning>
  Removing system recommendations means Tether will regenerate them if the need still exists. If you want to suppress a recommendation, consider adjusting safety stock or forecast settings.
</Warning>

## Alert Levels Explained

| Level | Criteria | Recommended Action |
|-------|----------|-------------------|
| **Urgent** | Stockout within lead time | Place order immediately |
| **Important** | Safety stock breach expected | Order within 1-2 days |
| **Normal** | Routine replenishment | Order per normal schedule |

## Integration with Transactions

When you create a draft from the supply plan:

1. **Draft Created** - Transaction is created in Draft status
2. **Editable** - You can still modify details
3. **Submit** - Submit to begin order processing
4. **In Progress** - Order is being fulfilled
5. **Completed** - Inventory is received

See [Transactions](/transactions/overview) for more on the transaction lifecycle.

## Troubleshooting

### No Recommendations Showing

**Possible causes:**
- No inventory issues projected
- Settings not configured (suppliers, lead times)
- Forecast data missing

**Solutions:**
1. Check inventory alerts on Overview tab
2. Verify supplier capabilities are configured
3. Ensure demand forecasts exist

### Recommendations Seem Wrong

**Possible causes:**
- Incorrect lead times
- Forecast is inaccurate
- Safety stock settings off

**Solutions:**
1. Verify supplier lead times
2. Review demand forecast accuracy
3. Adjust safety stock thresholds
