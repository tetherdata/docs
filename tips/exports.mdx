# Data Exports

Export data from Tether to use in spreadsheets, reports, and other tools.

## Why Export Data?

| Use Case | Description |
|----------|-------------|
| **Reporting** | Create custom reports |
| **Analysis** | Analyze in Excel or other tools |
| **Sharing** | Share with stakeholders |
| **Backup** | Keep local copies |
| **Integration** | Use in other systems |

## Export Formats

| Format | Best For |
|--------|----------|
| **CSV** | Spreadsheets, data import |
| **Excel (.xlsx)** | Formatted reports, analysis |
| **PDF** | Sharing, printing |

## Large Exports

### Handling Large Data Sets

For large exports:

| Size | Approach |
|------|----------|
| **< 10,000 rows** | Direct download |
| **10,000 - 100,000** | May take longer |
| **> 100,000** | Consider filtering |

### Tips for Large Exports

1. **Filter first** - Export only what you need
2. **Choose CSV** - Faster than Excel for large files
3. **Wait for completion** - Don't navigate away
4. **Check file** - Verify all data exported

## Troubleshooting

### Export Failed

**Possible causes:**
- Too much data
- Network timeout
- Browser issue

**Solutions:**
1. Try smaller export
2. Use CSV instead of Excel
3. Try different browser
4. Check network connection

### Missing Data in Export

**Possible causes:**
- Filters excluding data
- Field not selected
- Permission restrictions

**Solutions:**
1. Check active filters
2. Verify field selection
3. Confirm you have access

### File Won't Open

**Possible causes:**
- Large file size
- Corrupted download
- Wrong application

**Solutions:**
1. Re-download file
2. Try different application
3. Open as text to check format
