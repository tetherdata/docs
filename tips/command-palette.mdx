# Command Palette

The command palette provides quick access to any feature in Tether. Use it to navigate, search, and execute actions without leaving the keyboard.

## Opening the Command Palette

| Method | Action |
|--------|--------|
| **Keyboard** | `Cmd + K` (Mac) or `Ctrl + K` (Windows) |
| **Click** | Click search icon in header |

## Using the Command Palette

### Basic Navigation

<Steps>
  <Step title="Open Palette">
    Press `Cmd + K` or `Ctrl + K`
  </Step>

  <Step title="Type Search">
    Start typing what you want
  </Step>

  <Step title="Select Result">
    Use arrow keys or click to select
  </Step>

  <Step title="Execute">
    Press Enter to navigate or execute
  </Step>
</Steps>

### Search Types

The command palette searches across:

| Type | Examples |
|------|----------|
| **Pages** | SKUs, Inventory, Forecasting |
| **Actions** | Create SKU, Run Simulation |
| **Records** | Specific SKUs, orders, suppliers |
| **Settings** | User access, terminology |

## Command Categories

### Navigation Commands

Quickly jump to any page:

| Command | Navigates To |
|---------|--------------|
| `skus` | SKU list |
| `collections` | Collections page |
| `inventory` | Inventory dashboard |
| `forecast` | Demand forecasting |
| `transactions` | Transaction list |
| `settings` | Settings page |

### Action Commands

Execute common actions:

| Command | Action |
|---------|--------|
| `create sku` | Open SKU creation |
| `create collection` | Open collection creation |
| `create transaction` | Start new transaction |
| `run simulation` | Execute supply simulation |
| `export` | Open export options |

### Search Commands

Find specific records:

| Command | Finds |
|---------|-------|
| `sku:` + name | Specific SKU |
| `collection:` + name | Specific collection |
| `order:` + number | Specific transaction |
| `supplier:` + name | Specific supplier |

## Tips for Effective Use

### Fuzzy Matching

You don't need exact matches:

| Type | Matches |
|------|---------|
| `inv` | Inventory |
| `sku cre` | Create SKU |
| `fore dash` | Forecasting Dashboard |

### Recent Items

Command palette shows:

- Recent pages visited
- Recent searches
- Recent actions

### Keyboard Navigation

| Key | Action |
|-----|--------|
| `↑` `↓` | Navigate results |
| `Enter` | Select highlighted |
| `Esc` | Close palette |
| `Tab` | Autocomplete |

## Advanced Features

### Filtering Results

Narrow results by type:

| Prefix | Filter |
|--------|--------|
| `>` | Actions only |
| `@` | Records only |
| `#` | Pages only |

### Contextual Commands

Commands change based on current page:

| Current Page | Available Commands |
|--------------|-------------------|
| **SKU List** | Filter, sort, export |
| **Forecast** | Edit, compare, export |
| **Transaction** | Update status, add items |

## Troubleshooting

### Command Not Found

**Possible causes:**
- Misspelling
- Feature not available
- Permission restrictions

**Solutions:**
1. Check spelling
2. Try alternative phrasing
3. Verify you have access

### Slow Results

**Possible causes:**
- Large dataset
- Network latency
- Complex search

**Solutions:**
1. Be more specific
2. Use filters
3. Wait for results to load
