# Keyboard Shortcuts

Master keyboard shortcuts to work faster in Tether. These shortcuts help you navigate, edit, and manage data without reaching for the mouse.

## Global Shortcuts

Available throughout Tether:

| Shortcut | Action |
|----------|--------|
| `Cmd/Ctrl + K` | Open command palette |
| `Cmd/Ctrl + S` | Save current changes |
| `Esc` | Close modal/cancel action |
| `?` | Show keyboard shortcuts help |

## Navigation Shortcuts

### Page Navigation

| Shortcut | Navigates To |
|----------|--------------|
| `G` then `S` | SKUs |
| `G` then `C` | Collections |
| `G` then `I` | Inventory |
| `G` then `F` | Forecasting |
| `G` then `T` | Transactions |
| `G` then `A` | Analytics |

### Within Pages

| Shortcut | Action |
|----------|--------|
| `J` / `K` | Move down/up in lists |
| `Enter` | Open selected item |
| `Backspace` | Go back |
| `Home` / `End` | Jump to top/bottom |

## Table Shortcuts

### Selection

| Shortcut | Action |
|----------|--------|
| `Space` | Toggle row selection |
| `Shift + Click` | Select range |
| `Cmd/Ctrl + A` | Select all |
| `Cmd/Ctrl + Shift + A` | Deselect all |

### Navigation

| Shortcut | Action |
|----------|--------|
| `↑` `↓` | Move between rows |
| `←` `→` | Move between columns |
| `Tab` | Next cell |
| `Shift + Tab` | Previous cell |

### Editing

| Shortcut | Action |
|----------|--------|
| `Enter` | Edit selected cell |
| `Esc` | Cancel edit |
| `Delete` | Clear cell value |
| `Cmd/Ctrl + Z` | Undo |

## Customizing Shortcuts

### Viewing All Shortcuts

Press `?` anywhere in Tether to see:

- All available shortcuts
- Context-specific shortcuts
- Current page shortcuts

### Shortcut Conflicts

If shortcuts conflict with browser:

1. Check browser extensions
2. Disable conflicting shortcuts
3. Use alternative key combinations

## Troubleshooting

### Shortcuts Not Working

**Possible causes:**
- Browser extension conflict
- Focus not on application
- Modal is open

**Solutions:**
1. Click in the application area
2. Close any open modals
3. Check for conflicts

### Wrong Action Triggered

**Possible causes:**
- Similar shortcut sequences
- Modifier key stuck
- Page-specific shortcut

**Solutions:**
1. Release all keys
2. Check current context
3. Use command palette instead
